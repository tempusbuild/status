# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 3.0, the canonical text of which is at
<https://www.contributor-covenant.org/version/3/0/code_of_conduct/>.

By participating (issues, pull requests, discussions) you agree to uphold it.

To report unacceptable behavior, use GitHub's
[private vulnerability reporting](https://github.com/tempusbuild/status/security/advisories/new)
on this repository; reports are handled confidentially.
