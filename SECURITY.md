# Security policy

This repository hosts the **public status page** for tempus.build. It runs entirely on
GitHub Actions, Issues, and Pages, holds no application code and no customer data, and
keeps no secrets in the repository.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

- For an issue **in this repository** (e.g. a workflow or supply-chain concern), use
  GitHub's [private vulnerability reporting](https://github.com/tempusbuild/status/security/advisories/new).
- For an issue in the **tempus.build product or infrastructure**, use the private
  reporting channel on the relevant tempus.build repository — not this one.

We aim to acknowledge a report within a few business days. Please give us a reasonable
window to remediate before any public disclosure.
